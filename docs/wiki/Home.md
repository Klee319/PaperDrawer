# PaperDrawers Wiki

**PaperDrawers** は、Paper サーバー向けの大容量ストレージプラグインです。
ブロックの前面をクリックするだけで、大量のアイテムを1箇所に格納・取り出しできます。

## 主な特徴

- **大容量ストレージ** -- Tier 1 (32スタック) から Tier 7 (2048スタック) まで段階的にアップグレード可能
- **直感的な操作** -- 右クリックで挿入、左クリックで取り出し。チェストを開く必要なし
- **一括操作** -- Shift+右クリックでスタック全体を挿入、ダブル右クリックでインベントリ内の同種アイテムを全投入
- **ホッパー連携** -- バニラのホッパーと連携してアイテムの自動搬入・搬出が可能
- **仕分けドロワー** -- 隣接するホッパーからアイテムを自動で吸い取る高機能ドロワー
- **ボイドドロワー** -- 不要なアイテムを消滅させるゴミ箱機能
- **ドロワーキー** -- スロットを特定のアイテムにロックし、仕分けシステムを構築可能
- **Bedrock/Geyser 対応** -- 統合版プレイヤーにも対応

## 対応環境

| 項目 | 要件 |
|------|------|
| サーバー | Paper 1.21 以降 |
| オプション | Floodgate (統合版対応) |
