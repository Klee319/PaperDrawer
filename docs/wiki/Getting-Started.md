# はじめに

## インストール

1. [Paper 1.21](https://papermc.io/) 以降のサーバーを用意します
2. `PaperDrawers.jar` をサーバーの `plugins/` フォルダに配置します
3. サーバーを起動（または再起動）します
4. `plugins/PaperDrawers/` フォルダに設定ファイルが自動生成されます

> 統合版プレイヤーへの対応が必要な場合は、[Floodgate](https://geysermc.org/) プラグインも併せて導入してください。PaperDrawers は Floodgate を自動検出し、統合版プレイヤーにはアーマースタンドベースの表示にフォールバックします。

## 最初のドロワーを設置する

### 1. ドロワーをクラフトする

最も基本的な **Tier 1 ドロワー（基本のドロワー）** は、木材と チェストで作れます。

```
[木材] [木材] [木材]
[木材] [チェスト] [木材]
[木材] [木材] [木材]
```

木材はすべての種類（オーク、シラカバ、マツ、ジャングル、アカシア、ダークオーク、マングローブ、サクラ、竹）が使用可能です。

### 2. ドロワーを設置する

クラフトしたドロワーアイテムを地面やブロックに向けて右クリックで設置します。ドロワーの正面はプレイヤーの方を向きます（樽と同じ見た目のブロックです）。

### 3. アイテムを格納する

ドロワーの **正面** を操作してアイテムの出し入れを行います。

## 基本操作

### アイテムを入れる（右クリック）

| 操作 | 動作 |
|------|------|
| **右クリック** | 手持ちのアイテムを **1個** 挿入 |
| **Shift + 右クリック** | 手持ちの **スタック全体** を挿入 |
| **ダブル右クリック** | インベントリ内の **同種アイテムを全て** 挿入 |
| **素手で右クリック** | ドロワーに格納中の同種アイテムをインベントリから自動検索して挿入 |

### アイテムを取り出す（左クリック）

ドロワーの正面を左クリックすると、格納されているアイテムが **1個** 手元に出てきます。

> **注意:** 正面の縁（端5%）を左クリックすると、通常のブロック破壊として扱われます。アイテムを取り出すにはドロワー正面の中央部分をクリックしてください。

## ドロワーの表示

設置されたドロワーの正面には、以下の情報が表示されます。

- 格納されているアイテムのアイコン
- アイテムの個数（デフォルトでは `1K`, `1M` のようなコンパクト表記）

表示はリアルタイムで更新され、最大32ブロック離れた距離から確認できます。
