# ドロワーキー

ドロワーキーは、ドロワーのスロットを特定のアイテムに **ロック** するための特殊アイテムです。ロックされたスロットはアイテムが空になっても指定されたアイテムのみを受け付けるため、自動仕分けシステムの構築に不可欠です。

## 見た目

ドロワーキーは **トリップワイヤーフック** の見た目をしたアイテムで、金色の名前「ドロワーキー」と使い方の説明が付いています。

## クラフトレシピ

1回のクラフトで **3個** 作成できます。

```
+---+---+---+
|   | G |   |
+---+---+---+
|   | S |   |
+---+---+---+
|   | S |   |
+---+---+---+

G = 金塊 (GOLD_NUGGET)
S = 棒 (STICK)
```

コマンドで入手する場合: `/drawer key [個数]`

## 使い方

### スロットをロックする

1. ドロワーにロックしたいアイテムを入れます（最低1個）
2. ドロワーキーを手に持ちます
3. ドロワーの正面を **右クリック** します
4. 「ドロワーを「(アイテム名)」にロックしました！」と表示されます

### スロットのロックを解除する

1. ドロワーキーを手に持ちます
2. ロック済みのドロワーの正面を **右クリック** します
3. 「ドロワーのロックを解除しました！」と表示されます

### ロックできない場合

- スロットが空の状態ではロックできません
- 先にアイテムを入れてからロックしてください
- 「空のスロットはロックできません。先にアイテムを入れてください！」と表示されます

## ロックされたスロットの動作

| 状態 | 動作 |
|------|------|
| ロック済み + アイテムあり | ロックされたアイテムと同じ種類のみ受け入れ |
| ロック済み + アイテムなし | ロックされたアイテムと同じ種類のみ受け入れ（空でも種類を記憶） |
| ロックなし + アイテムあり | 格納中のアイテムと同じ種類のみ受け入れ |
| ロックなし + アイテムなし | 任意のアイテムを受け入れ |

## 仕分けシステムでの活用

ドロワーキーの真価は [仕分けドロワー](Special-Drawers.md) と組み合わせた時に発揮されます。

仕分けドロワーは **設定済みスロット**（アイテムが入っているか、ロックされているスロット）のみホッパーからアイテムを吸い取ります。ドロワーキーでスロットをロックしておけば、アイテムが空になっても自動収集が継続します。

### 仕分けシステムの構築例

1. 仕分けドロワーを複数設置
2. 各ドロワーに仕分けたいアイテムを1個ずつ入れる
3. ドロワーキーで各スロットをロック
4. ホッパーチェーンで接続

```
[チェスト]
    ↓ (ホッパー)
[仕分けドロワー: 丸石 (ロック済)]
    ↓ (ホッパー)
[仕分けドロワー: 鉄鉱石 (ロック済)]
    ↓ (ホッパー)
[仕分けドロワー: 石炭 (ロック済)]
    ↓ (ホッパー)
[ボイドドロワー] ← 仕分け対象外を処分
```

これにより、アイテムが空になってもロックが維持され、次にそのアイテムが流れてきた時に自動的に仕分けられます。
