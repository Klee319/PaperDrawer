# クラフトレシピ

PaperDrawers の全クラフトレシピを掲載しています。全てのレシピは `recipes.yml` で個別に有効/無効を切り替えられます。

---

## ドロワーキー

ドロワースロットのロック・アンロックに使用する特殊アイテムです。1回のクラフトで **3個** 作成できます。

```
+---+---+---+
|   | G |   |
+---+---+---+
|   | S |   |
+---+---+---+
|   | S |   |
+---+---+---+

G = 金塊 (GOLD_NUGGET)
S = 棒 (STICK)
```

詳細は [ドロワーキー](Drawer-Key.md) を参照。

---

## Tier 1 -- 基本のドロワー

容量: 32 スタック (2,048 個)

```
+---+---+---+
| P | P | P |
+---+---+---+
| P | C | P |
+---+---+---+
| P | P | P |
+---+---+---+

P = 木材 (任意の種類の板材)
C = チェスト (CHEST)
```

> **ポイント:** 木材はオーク、シラカバ、マツ、ジャングル、アカシア、ダークオーク、マングローブ、サクラ、竹のいずれでも可。異なる種類の木材を混ぜてもクラフト可能です。

## Tier 2 -- 銅のドロワー

+---+---+---+

## Tier 7 -- クリエイティブドロワー (デフォルト無効)

容量: 2,048 スタック (131,072 個)

```
+---+---+---+
| N | N | N |
+---+---+---+
| N | D | N |
+---+---+---+
| N | N | N |
+---+---+---+

N = ネザースター (NETHER_STAR)
D = Tier 6 ドロワー
```

> **注意:** このレシピはデフォルトで **無効** です。`recipes.yml` で `tier-7.enabled: true` に設定すると有効化されます。

## ボイドドロワー

投入したアイテムを消滅させるゴミ箱ドロワーです。

```
+---+---+---+
| O | E | O |
+---+---+---+
| E | C | E |
+---+---+---+
| O | E | O |
+---+---+---+

O = 黒曜石 (OBSIDIAN)
E = エンダーパール (ENDER_PEARL)
C = チェスト (CHEST)
```

---

## 仕分けドロワー (Tier 1～7)

仕分けドロワーは、対応するティアの通常ドロワーをコンパレータとホッパーで挟んでクラフトします。全ティア共通のパターンです。

```
+---+---+---+
|   | C |   |
+---+---+---+
|   | D |   |
+---+---+---+
|   | H |   |
+---+---+---+

C = コンパレータ (COMPARATOR)
D = 対応ティアのドロワー
H = ホッパー (HOPPER)
```

| レシピ | 中央のドロワー (D) | デフォルト |
|--------|-------------------|:---------:|
| 仕分け Tier 1 | 基本のドロワー (Tier 1) | 有効 |
| 仕分け Tier 2 | 銅のドロワー (Tier 2) | 有効 |
| 仕分け Tier 3 | 鉄のドロワー (Tier 3) | 有効 |
| 仕分け Tier 4 | 金のドロワー (Tier 4) | 有効 |
| 仕分け Tier 5 | ダイヤのドロワー (Tier 5) | 有効 |
| 仕分け Tier 6 | ネザライトのドロワー (Tier 6) | 有効 |
| 仕分け Tier 7 | クリエイティブドロワー (Tier 7) | **無効** |

---

## レシピの無効化・カスタマイズ

各レシピは `recipes.yml` で個別に制御できます。

```yaml
recipes:
  tier-1:
    enabled: false  # Tier 1 のクラフトを無効化
```

詳細は [設定ファイル](Configuration.md) を参照してください。
