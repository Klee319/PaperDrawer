# よくある質問 (FAQ)

## アイテムが消えた / 増えた

**Q: ホッパーを繋いだらアイテムが消えました。**

A: PaperDrawers はバニラのホッパー転送をブロックし、独自の転送システムを使用しています。以下を確認してください。

- ホッパーの **排出口がドロワーの方向を向いている** か確認（横からの搬入はホッパーの向きが重要）
- ドロワーの下にホッパーを置く場合、ホッパーが **レッドストーン信号で無効化されていない** か確認
- `config.yml` の `hopper.enabled` が `true` になっているか確認

---

## クラフトに関する問題

**Q: レシピ通りに配置してもクラフトできません。**

A: 以下を確認してください。

- `config.yml` で該当レシピの `enabled` が `true` になっているか
- Tier 7 および 仕分け Tier 7 はデフォルトで **無効** です
- Tier 2 以降のアップグレードには、**前のティアのドロワーアイテム** が必要です。設置してあるドロワーブロックではなく、インベントリ内のドロワーアイテムを使ってください
- 設定変更後は `/drawer reload` を実行してください

**Q: 木材の種類を混ぜてもクラフトできますか？**

A: はい。Tier 1 のレシピでは `TAG_PLANKS`（板材タグ）を使用しており、全種類の木材（オーク、シラカバ、マツ、ジャングル、アカシア、ダークオーク、マングローブ、サクラ、竹）が使用可能です。異なる種類を混ぜてもクラフトできます。

## 仕分けドロワーに関する問題

**Q: 仕分けドロワーがホッパーからアイテムを吸い取りません。**

A: 仕分けドロワーは **設定済みスロット** のみアイテムを受け入れます。

1. ドロワーにアイテムを1個以上入れるか、[ドロワーキー](Drawer-Key.md) でスロットをロックしてください
2. ホッパーが隣接しているか確認（上下左右前後の6方向）
3. ホッパーがレッドストーン信号で無効化されていないか確認

**Q: 仕分けドロワーのアイテムが空になると吸い取りが止まります。**

A: ドロワーキーでスロットをロックしてください。ロックされたスロットはアイテムが空になっても設定が維持されるため、対象アイテムが流れてくれば自動的に吸い取りを再開します。

## Bedrock / Geyser 対応

**Q: 統合版 (Bedrock) プレイヤーでもドロワーは使えますか？**

A: はい。[Floodgate](https://geysermc.org/) プラグインを導入すると、PaperDrawers は自動的に統合版プレイヤーを検出します。

- `config.yml` の `bedrock.fallback-to-armor-stand` が `true` (デフォルト) の場合、統合版プレイヤーには ItemDisplay の代わりに ArmorStand ベースの表示が使われます
- Java 版プレイヤーには影響ありません
- `/drawer info` コマンドで Floodgate が検出されているか確認できます

## パフォーマンスに関する問題

**Q: ドロワーが多くなるとサーバーが重くなります。**

A: 以下の設定を調整してみてください。

### ホッパー処理の間隔を広げる

```yaml
hopper:
  pull-interval-ticks: 8  # デフォルトの 4 から 8 に変更（バニラと同速度）
```

### 表示の更新間隔を広げる

```yaml
display:
  update-interval: 10  # デフォルトの 5 から 10 に変更
  max-distance: 16     # デフォルトの 32 から 16 に変更
```

### キャッシュの有効化を確認

```yaml
performance:
  cache:
    enabled: true      # 必ず true にしておく
    max-size: 1000     # ドロワー数に応じて調整
```

### 非同期保存を有効化

```yaml
performance:
  async-save:
    enabled: true      # メインスレッドのブロッキングを減らす
```

### メトリクスログで状態を確認

```yaml
performance:
  metrics-log-interval-minutes: 5  # コンソールに定期的にパフォーマンス情報を出力
```

`/drawer info` コマンドでキャッシュの統計情報や保留中の保存数を確認できます。

## その他

**Q: ドロワーを壊すとアイテムはどうなりますか？**

A: ドロワーを壊すと、格納されていたアイテムがドロップします。

**Q: ドロワーキーはどこで入手できますか？**

A: クラフトするか、`/drawer key` コマンドで入手できます。クラフトレシピは金塊と棒で、1回で3個作成できます。詳しくは [ドロワーキー](Drawer-Key.md) を参照してください。

**Q: コマンドのエイリアスはありますか？**

A: `/drawer` の他に `/drawers` と `/pd` が使用可能です。

**Q: プラグインのバージョンを確認するには？**

A: `/drawer info` コマンドで確認できます。
