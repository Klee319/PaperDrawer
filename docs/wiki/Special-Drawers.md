# 特殊なドロワー

通常のティア制ドロワーに加え、PaperDrawers には特殊な機能を持つドロワーが2種類あります。

## ボイドドロワー

**投入されたアイテムを全て消滅させる** ゴミ箱機能を持つドロワーです。

### 基本情報

| 項目 | 内容 |
|------|------|
| 表示名 | ボイドドロワー |
| カラー | &8（暗い灰色） |
| 見た目 | ジュークボックスと同じブロック |
| 格納容量 | なし（アイテムは即座に消滅） |

### 使い方

1. ボイドドロワーを設置します
2. 正面を右クリックでアイテムを投入します
3. 投入されたアイテムは即座に消滅します

> **警告:** ボイドドロワーに投入したアイテムは **復元できません**。誤って貴重なアイテムを投入しないよう注意してください。

### ホッパーとの連携

ボイドドロワーはホッパーからのアイテム搬入にも対応しています。ホッパーをボイドドロワーに向けて設置すると、ホッパー内のアイテムが自動的に消滅します。これはゴミ処理システムの構築に便利です。

### クラフトレシピ

```
[黒曜石]     [エンダーパール] [黒曜石]
[エンダーパール] [チェスト]     [エンダーパール]
[黒曜石]     [エンダーパール] [黒曜石]
```

詳細は [クラフトレシピ](Crafting-Recipes.md) を参照してください。

## 仕分けドロワー

**隣接するホッパーからアイテムを自動的に吸い取る** 高機能ドロワーです。通常のドロワーと同じ容量を持ちながら、自動収集機能が追加されています。

### 基本情報

| 項目 | 内容 |
|------|------|
| ティア | Tier 1～7（通常ドロワーと同じ容量体系） |
| 自動収集範囲 | 隣接する6方向（上下左右前後） |
| 対象コンテナ | ホッパー |

### 仕分けの仕組み

仕分けドロワーは以下の条件で動作します。

1. **設定済みスロット** のみアイテムを受け入れます
   - アイテムが既に格納されているスロット
   - ドロワーキーでロックされたスロット
2. 隣接する6方向のホッパーを走査します
3. ホッパー内にスロットの受入可能なアイテムがあれば、1個ずつ吸い取ります

つまり、仕分けドロワーを使うには:

1. まず手動でアイテムを1個入れるか、[ドロワーキー](Drawer-Key.md) でスロットをロックする
2. 隣接するホッパーに対象アイテムが流れてくると自動的に吸い取る

### 使い方の例

#### 基本的な仕分けライン

```
[アイテム源] → [ホッパー] → [仕分けドロワー]
                              (丸石にロック済み)
```

1. 仕分けドロワーを設置
2. 丸石を1個入れるか、ドロワーキーで丸石にロック
3. 隣接するホッパーに丸石が流れてくると自動で吸い取り

#### 複数素材の仕分け

複数の仕分けドロワーを並べて、ホッパーチェーンで接続:

```
[ホッパー] → [仕分けドロワー: 丸石]
     ↓
[ホッパー] → [仕分けドロワー: 鉄鉱石]
     ↓
[ホッパー] → [仕分けドロワー: 石炭]
     ↓
[ボイドドロワー] ← 余りを処分
```

### レッドストーン制御

レッドストーン信号を受けたホッパーは無効化されるため、仕分けドロワーはそのホッパーからアイテムを吸い取りません。これはバニラのホッパー動作と同じです。

### クラフトレシピ

仕分けドロワーは、対応するティアの通常ドロワーをコンパレータとホッパーで挟んでクラフトします。

```
[  ] [コンパレータ] [  ]
[  ] [ドロワー]     [  ]
[  ] [ホッパー]     [  ]
```

全ティアのレシピは [クラフトレシピ](Crafting-Recipes.md) を参照してください。

> **注意:** 仕分け Tier 7 のレシピは通常の Tier 7 と同様に **デフォルトで無効** です。
