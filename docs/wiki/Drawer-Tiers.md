# ドロワーのティア

PaperDrawers は 7段階のティア（等級）を持ち、上位のティアほど大容量のストレージを提供します。

## ティア一覧

| ティア | 表示名 | 容量 (スタック) | 容量 (個数) | アップグレード素材 |
|:------:|--------|:--------------:|:-----------:|-------------------|
| 1 | 基本のドロワー | 32 | 2,048 | 木材 + チェスト |
| 2 | 銅のドロワー | 64 | 4,096 | 銅インゴット + Tier 1 ドロワー |
| 3 | 鉄のドロワー | 128 | 8,192 | 鉄インゴット + Tier 2 ドロワー |
| 4 | 金のドロワー | 256 | 16,384 | 金インゴット + Tier 3 ドロワー |
| 5 | ダイヤのドロワー | 512 | 32,768 | ダイヤモンド + Tier 4 ドロワー |
| 6 | ネザライトのドロワー | 1,024 | 65,536 | ネザライトインゴット + Tier 5 ドロワー |
| 7 | クリエイティブドロワー | 2,048 | 131,072 | ネザースター + Tier 6 ドロワー |

> **注意:** 容量の「個数」は、1スタック = 64個で計算した場合の値です。エンダーパールなど最大スタック数が異なるアイテムでは実際の個数は変わりますが、スタック数は同じです。

## アップグレードの仕組み

ドロワーのティアを上げるには、現在のティアのドロワーを素材として使い、次のティアをクラフトします。

```
アップグレードの流れ:

Tier 1 (木材+チェスト)
  ↓ + 銅インゴット
Tier 2
  ↓ + 鉄インゴット
Tier 3
  ↓ + 金インゴット
Tier 4
  ↓ + ダイヤモンド
Tier 5
  ↓ + ネザライトインゴット
Tier 6
  ↓ + ネザースター
Tier 7 (クリエイティブ)
```

各レシピの詳細は [クラフトレシピ](Crafting-Recipes.md) を参照してください。

## Tier 7 (クリエイティブドロワー) について

Tier 7 のクラフトレシピは **デフォルトで無効** になっています。サーバー管理者が `recipes.yml` で有効化することで使用可能になります。

```yaml
recipes:
  tier-7:
    enabled: true  # false から true に変更
```

Tier 7 はネザースター8個と Tier 6 ドロワー1個という非常に高コストなレシピとなっています。

## 各ティアの色分け

ゲーム内ではティアごとに色が割り当てられています。

| ティア | カラーコード | 色 |
|:------:|:-----------:|:---:|
| 1 | &f | 白 |
| 2 | &6 | 金色 |
| 3 | &7 | 灰色 |
| 4 | &e | 黄色 |
| 5 | &b | 水色 |
| 6 | &4 | 暗い赤 |
| 7 | &d | ピンク |
